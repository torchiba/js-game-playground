# tic-tac-toe（○×ゲーム）
## 仕様

- 先攻が○×どちらのマークを使うか選びます
- 先攻後攻で9マスの中から好きなマスを選んで先に1列揃ったほうが勝利です（先攻も後攻も自分で操作します）
- 1列揃わない状態で9マスがすべて埋まってしまうとゲームオーバーです
- ゲームオーバーもしくは1列揃うとゲーム終了です
- ゲーム終了画面でもう一度プレイするボタンを選ぶと再度ゲームが出来ます
